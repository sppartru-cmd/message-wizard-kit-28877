# Подробный гайд по запуску проекта локально

## Требования

### Для фронтенда:
- Node.js версии 18 или выше
- npm или bun (менеджер пакетов)

### Для бэкенда:
- Python 3.8 или выше
- pip (менеджер пакетов Python)

## Шаг 1: Скачивание проекта

1. Скачайте все файлы проекта к себе на компьютер
2. Разархивируйте если нужно
3. Откройте папку проекта в терминале/командной строке

## Шаг 2: Установка и запуск фронтенда

### Windows:
```cmd
# Установка зависимостей
npm install

# Запуск сервера разработки
npm run dev
```

### Linux/macOS:
```bash
# Установка зависимостей
npm install

# Запуск сервера разработки
npm run dev
```

После запуска фронтенд будет доступен по адресу: `http://localhost:5173`

## Шаг 3: Установка и запуск бэкенда (Python)

### Windows:

1. Откройте **новый** терминал (не закрывая терминал с фронтендом)
2. Перейдите в папку python-backend:
```cmd
cd python-backend
```

3. Создайте виртуальное окружение (рекомендуется):
```cmd
python -m venv venv
venv\Scripts\activate
```

4. Установите зависимости:
```cmd
pip install -r requirements.txt
```

5. Запустите сервер:
```cmd
python api_server.py
```

### Linux/macOS:

1. Откройте **новый** терминал (не закрывая терминал с фронтендом)
2. Перейдите в папку python-backend:
```bash
cd python-backend
```

3. Создайте виртуальное окружение (рекомендуется):
```bash
python3 -m venv venv
source venv/bin/activate
```

4. Установите зависимости:
```bash
pip install -r requirements.txt
```

5. Запустите сервер:
```bash
python3 api_server.py
```

Бэкенд будет доступен по адресу: `http://localhost:5000`

## Шаг 4: Настройка WhatsApp

1. Откройте браузер и перейдите на `http://localhost:5173`
2. Зайдите в раздел "Профили"
3. Создайте новый профиль
4. WhatsApp Web откроется в браузере - отсканируйте QR-код
5. После успешной авторизации профиль будет готов к использованию

## Проверка работоспособности

### Тест фронтенда:
- Откройте `http://localhost:5173` в браузере
- Должен загрузиться интерфейс приложения

### Тест бэкенда:
- Откройте в браузере `http://localhost:5000/api/profiles`
- Должен вернуться JSON с профилями (может быть пустой массив)

### Тест связи фронтенд-бэкенд:
- В интерфейсе создайте профиль
- Если профиль создался успешно - связь работает

## Распространенные проблемы

### Ошибка "npm: command not found" или "node: command not found"
- Установите Node.js с официального сайта: https://nodejs.org/
- После установки перезапустите терминал

### Ошибка "python: command not found"
- **Windows**: Установите Python с официального сайта: https://www.python.org/
  - При установке обязательно поставьте галочку "Add Python to PATH"
- **Linux**: `sudo apt install python3 python3-pip`
- **macOS**: `brew install python3`

### Ошибка "Порт уже занят" (Port already in use)
- Закройте другие приложения, использующие порты 5173 или 5000
- Или измените порт в файлах конфигурации

### Ошибка импорта модулей Python
- Убедитесь, что виртуальное окружение активировано
- Переустановите зависимости: `pip install -r requirements.txt --force-reinstall`

### WhatsApp не подключается
- Проверьте что бэкенд запущен и доступен
- Проверьте консоль браузера на наличие ошибок
- Попробуйте перезагрузить страницу и переподключиться

## Структура проекта

```
project/
├── src/                    # Исходный код фронтенда (React)
│   ├── components/        # Компоненты React
│   ├── pages/            # Страницы приложения
│   └── lib/              # Вспомогательные функции
├── python-backend/        # Бэкенд на Python
│   ├── api_server.py     # Основной сервер
│   ├── whatsapp_sender.py # Логика WhatsApp
│   └── requirements.txt   # Зависимости Python
└── package.json          # Зависимости фронтенда
```
